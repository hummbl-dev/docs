## What's New

- Governed control plane for AI agent systems
- Registry-first skill management
- Deterministic policy routing
- Auditable execution across any LLM runner
- 75+ skills with SHA-256 integrity
- 10 LLM backend adapters (Claude, Codex, Gemini, Grok, Ollama, etc.)
- JSON policy DSL for deterministic compilation
- Vendor-agnostic execution layer

cd hummbl-agent
corepack enable
pnpm install
