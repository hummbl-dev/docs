# Contributing to Evidence Gate

Thank you for your interest in contributing to Evidence Gate.

## Guidelines

- **Python packages**: stdlib-only for runtime dependencies.
- **Commit format**: Conventional Commits (`feat:`, `fix:`, `docs:`, `refactor:`, `test:`, `chore:`).
- **Pull requests**: Branch from `main`, ensure fixtures pass, squash merge.
- **License**: All contributions are licensed under Apache 2.0.

## Process

1. Fork the repository
2. Create a feature branch (`feat/your-feature`)
3. Make your changes with fixture tests
4. Run the fixture harness (`python scripts/run_rule_fixtures.py`)
5. Open a pull request against `main`
