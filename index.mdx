Hummbl is open-source AI governance infrastructure shipped as importable Python libraries. Instead of a SaaS dashboard you connect to after the fact, Hummbl gives you kill switches, audit logs, delegation tokens, and circuit breakers that live directly inside your agent runtime — no accounts, no external services, no third-party dependencies.

## Get up and running

The following steps take you from a fresh environment to a fully governed agent with an audit trail, a kill switch, and scoped delegation tokens.

<Steps>
  <Step title="Install the library">
    Install `hummbl-governance` from PyPI. Python 3.11 or later is required. No third-party packages are pulled in.

    ```bash
    pip install hummbl-governance
    ```
  </Step>
  <Step title="Wire the kill switch first">
    The kill switch is the first thing your agent checks on every run. If it is engaged, the agent halts before taking any action.

    ```python
    from pathlib import Path
    from hummbl_governance import KillSwitch

    state = Path("_state")
    state.mkdir(exist_ok=True)

    ks = KillSwitch(state_dir=state)
    if ks.engaged():
        raise SystemExit(f"kill switch active: {ks.get_status()}")
    ```
  </Step>
  <Step title="Issue a delegation token">
    Delegation tokens define exactly what your agent is allowed to do. Every operation is checked against the token's scope before it executes.

    ```python
    import os
    os.environ["ENABLE_IDP"] = "true"

    from hummbl_governance import DelegationTokenManager

    mgr = DelegationTokenManager()
    token = mgr.create_token(
        agent_id="research-agent-1",
        scope=["read:web", "write:drafts"],
        ttl=3600,
        chain_depth=1,
    )
    ```
  </Step>
  <Step title="Log every action to the governance bus">
    The governance bus is your tamper-evident audit log. Write a record for every action your agent takes — approvals, denials, errors, and receipts.

    ```python
    from hummbl_governance import BusWriter

    bus = BusWriter(bus_path=state / "governance_bus.jsonl")

    bus.post(
        from_id="research-agent-1",
        to_id="all",
        msg_type="ACTION",
        message="fetching https://api.hummbl.io/health",
    )
    ```
  </Step>
</Steps>

<Note>
  `hummbl-governance` and `hummbl-base120` have **zero third-party runtime dependencies** — they rely entirely on the Python standard library. This means you can deploy them in air-gapped environments, government enclaves, and strict supply-chain contexts without any additional review or approval.
</Note>
